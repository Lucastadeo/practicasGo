1) create a new repository on the command line

echo "# practicasgo" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/Lucastadeo/practicasgo.git
git push -u origin main

git remote add origin https://github.com/Lucastadeo/practicasgo.git
git branch -M main
git push -u origin main
